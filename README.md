
# PEGADOCS 🖇️📑

## ¿Qué es Pegadocs?
Una herramienta colaborativa para ensamblar PDFs. El problema que resuelve: cuando una tarea grupal requiere que cada integrante entregue su sección, siempre alguien termina recolectando y uniendo los archivos manualmente. Pegadocs estructura ese proceso — cada miembro sube su parte, y el PDF final se ensambla en el navegador (similar a iLovePDF) con orden drag & drop antes de exportar.


## Stack Tecnológico

- Vite/Vue 3
- Nuxt UI v4
- Laravel 13
- Postgresql
- Storage (local por el momento)
